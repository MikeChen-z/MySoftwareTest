# 软件测试实验二 - 边界值测试

## 项目概述
本项目是《软件测试》实验二，主要涉及边界值分析测试方法的实践应用。

## 测试方法
- **一般边界值测试**: 对输入变量的边界进行系统测试
- **健壮性边界值测试**: 考虑越界情况，增加异常值测试
- **最坏情况一般边界值测试**: 测试多个变量边界的组合
- **最坏情况健壮性边界值测试**: 综合考虑多变量组合和越界场景

## 测试用例
测试用例位于 `app/src/test/resources/` 目录:
- `一般边界值测试.csv` - 一般边界值测试用例
- `健壮性边界值测试.csv` - 健壮性边界值测试用例
- `最坏情况一般边界值测试.csv` - 多变量边界组合
- `最坏情况健壮性边界值测试.csv` - 多变量+越界组合

## 运行方式

### 编译
```bash
cd 软件测试实验二
./gradlew build
```

### 运行测试
```bash
cd 软件测试实验二
./gradlew test
```

## 主要类说明
- `App.java` - 程序入口和核心测试逻辑
- `AppTest.java` - JUnit测试类，从CSV文件读取测试数据

## 技术栈
- 语言: Java
- 构建工具: Gradle
- 测试框架: JUnit 5 (推荐) / JUnit 4
- 数据格式: CSV (便于管理和维护大量测试用例)

## 学习重点
1. 理解边界值分析的含义和应用
2. 掌握参数化测试的实现方法
3. 学会从CSV文件读取测试数据
4. 理解一般/健壮性/最坏情况三种测试方法的区别
