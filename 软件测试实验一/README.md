# 软件测试实验一 - 三角形分类测试

## 项目概述
本项目是《软件测试》实验一，主要涉及三角形分类问题的多种测试方法实践。

## 测试方法
- **等价类测试**: 将可能的输入分为几个等价类，从每个等价类中选取代表性数据进行测试
- **边界值测试**: 在边界处进行测试，包括最小值、最大值、刚好超出边界等
- **健壮性测试**: 考虑异常和边界情况，提高测试的鲁棒性
- **最坏情况测试**: 测试最坏场景下的输入组合

## 测试用例
测试用例位于 `app/src/test/resources/` 目录:
- `三角形等价类测试用例.csv` - 等价类测试用例
- `三角形一般边界测试用例.csv` - 边界值测试用例
- `三角形健壮性测试用例.csv` - 健壮性测试用例
- `三角形健壮性最坏状况测试用例.csv` - 最坏情况测试用例
